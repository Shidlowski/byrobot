Наши возможности

Анализ производства

- технический аудит производства в сфере роботизации
- оценка производственных процессов для внедрения роботизации
- рекомендации по внедрению роботизированных комплексов/линий

Разработка проекта под ключ

- проработка,
- проектирование,
- шеф-монтаж,
- пусконаладка

Моделирование проекта. Подбор оборудования.

- моделирование компоновочного решения
- расчет цикла и загруженности оборудования
- подбор требуемых роботов
- подбор вспомогательного оборудования

Проектирование металлоконструкций, технологической оснастки, вспомогательных узлов

- разработка вспомогательных/технологических металлоконструкций
- разработка оснастки для установки и фиксации изделий
- разработка дополнительных механизмов периферийного оборудования

Программирование и запуск

- написание управляющих программ
- корректировка управляющих программ
- запуск роботизированных комплексов
- инструктаж обслуживающего персонала

Сервис. Техническая поддержка

- регламентное обслуживание оборудования
- замена смазки
- замена механических узлов
- замена кабелей

  • Обслуживание станков - загрузка/выгрузка деталей
  • Паллетирование
  • Укладка, упаковка, перемещение изделий
  • Сборка узлов
  • Измерение, контроль, сортировка
  • Нанесение покрытий
  • Моечные комплексы
  • Индивидуальные проекты
